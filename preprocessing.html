preprocessing

Data preprocessing and transformation for CyTOF data.

xcode_debarcode.preprocessing.filter_cells_intensity(adata, layer='log', method='rectangular', percentile=99.0, sum_low=1.0, sum_high=99.0, var_low=1.0, var_high=99.0, filter_or_flag='flag', filter_name=None, inplace=True, verbose=True)[source]

Filter cells by barcode channel intensity statistics.

Removes debris (low sum/variance) and doublets (high sum/variance) based on channel sum (L1 norm) and channel variance.

Parameters:

  • adata (AnnData) – Annotated data object with mapped barcode channels.

  • layer (str, optional, default 'log') – Data layer to use. None for adata.X.

  • method ({'rectangular', 'ellipsoidal'}, default 'rectangular') –

    Filtering method:

    • 'rectangular': Independent percentile thresholds on sum and variance.

    • 'ellipsoidal': Mahalanobis distance with robust covariance estimation.

  • percentile (float, default 99.0) – For 'ellipsoidal' method, keep cells within this percentile of Mahalanobis distance.

  • sum_low (float, optional, default 1.0) – For 'rectangular': lower percentile bound for channel sum. Cells below this percentile are removed. Set to None to disable.

  • sum_high (float, optional, default 99.0) – For 'rectangular': upper percentile bound for channel sum. Cells above this percentile are removed. Set to None to disable.

  • var_low (float, optional, default 1.0) – For 'rectangular': lower percentile bound for channel variance. Cells below this percentile are removed. Set to None to disable.

  • var_high (float, optional, default 99.0) – For 'rectangular': upper percentile bound for channel variance. Cells above this percentile are removed. Set to None to disable.

  • filter_or_flag ({'flag', 'filter'}, default 'flag') – 'flag' to add boolean column, 'filter' to remove cells.

  • filter_name (str, optional) – Custom name for this filtering run. Default: auto-generated.

  • inplace (bool, default True) – Modify adata in place.

  • verbose (bool, default True) – Print progress messages.

Returns:

AnnData with filtering results.

Return type:

AnnData

Notes

For rectangular method:

  • sum_low=1, sum_high=99 removes bottom 1% and top 1% by channel sum

  • var_low=1, var_high=99 removes bottom 1% and top 1% by variance

For ellipsoidal method:

  • Uses Minimum Covariance Determinant for robust covariance estimation

  • Filters based on Mahalanobis distance in (sum, variance) space

  • Better handles correlated sum/variance and non-axis-aligned outliers

xcode_debarcode.preprocessing.transform(adata, method='log', cofactor=10.0, layer_name=None, inplace=True, verbose=True)[source]

Apply transformation to intensity data.

Creates a new layer in adata with transformed values. The transformation is applied to ALL channels (barcode and non-barcode), but the function tracks which channels are barcode channels using adata.uns['barcode_channels'].

Parameters:

  • adata (AnnData) – Annotated data object (should have barcode channels mapped via map_channels).

  • method ({'log', 'arcsinh'}, default 'log') –

    Transformation method:

    • 'arcsinh': inverse hyperbolic sine, arcsinh(x/cofactor)

    • 'log': natural logarithm, log(x + 1)

  • cofactor (float, default 10.0) – Cofactor for arcsinh transformation.

  • layer_name (str, optional) – Name for the transformed layer. If None, auto-generates: 'arcsinh_cf{cofactor}' for arcsinh, 'log' for log.

  • inplace (bool, default True) – Modify adata in place.

  • verbose (bool, default True) – Print progress messages.

Returns:

Modified AnnData with new layer containing transformed data:

  • adata.layers[layer_name]: transformed data for all channels

  • adata.uns['transformations'][layer_name]: transformation metadata

Return type:

AnnData
