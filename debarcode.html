debarcode

Debarcoding methods for CyTOF barcode data.

xcode_debarcode.debarcode.debarcode(adata, method='pc_gmm', layer=None, n_init=5, min_int=0.1, prob_thresh=0.5, n_iter=2, thresholds=None, method_name=None, inplace=True, verbose=True)[source]

Apply debarcoding to assign barcodes to cells.

Parameters:

  • adata (AnnData) – Annotated data object with mapped barcode channels.

  • method ({'gmm', 'premessa', 'pc_gmm', 'manual'}, default 'pc_gmm') –

    Debarcoding method:

    • 'pc_gmm': Pattern-Constrained GMM (recommended for most datasets)

    • 'premessa': Iterative top-4 selection per block (for unimodal regime)

    • 'gmm': Gaussian Mixture Model per channel (exploration only; assignments not constrained to valid patterns)

    • 'manual': Fixed thresholds

  • layer (str, optional) – Layer to use for debarcoding. Recommend using transformed layer.

  • n_init (int, default 5) – Number of GMM initializations per channel. Used by: gmm, pc_gmm.

  • min_int (float, default 0.1) – Minimum intensity for GMM fitting. Used by: gmm, pc_gmm.

  • prob_thresh (float, default 0.5) – Probability threshold for ON classification. Used by: gmm.

  • n_iter (int, default 2) – Number of normalization iterations. Used by: premessa.

  • thresholds (list of float, optional) – Manual thresholds (one per channel). Required for 'manual' method. Used by: manual.

  • method_name (str, optional) – Custom name for this debarcoding run. If None, uses method name and auto-increments if exists (e.g., 'pc_gmm', 'pc_gmm_1').

  • inplace (bool, default True) – Modify adata in place.

  • verbose (bool, default True) – Print progress messages.

Returns:

Modified AnnData with debarcoding results:

  • adata.obs['{method_name}_assignment']: Assigned barcode patterns

  • adata.obs['{method_name}_confidence']: Raw method confidence score (not created for manual method). For PC-GMM: product of per-block posteriors. For GMM: product of per-channel posteriors. For PreMessa: minimum separation delta across blocks.

  • adata.uns['debarcoding']['{method_name}']: Method metadata

Return type:

AnnData

Examples

>>> adata = debarcode(adata, method='pc_gmm', layer='log')
>>> adata = debarcode(adata, method='premessa', layer='log')
>>> adata = debarcode(adata, method='manual', thresholds=[1.5]*27)
xcode_debarcode.debarcode.debarcoding_pipeline(adata, method='pc_gmm', transform_method='log', cofactor=10.0, n_init=5, min_int=0.1, prob_thresh=0.5, n_iter=2, thresholds=None, apply_intensity_filter=True, intensity_method='rectangular', intensity_percentile=99.0, intensity_sum_low=1.0, intensity_sum_high=99.0, intensity_var_low=1.0, intensity_var_high=99.0, intensity_filter_or_flag='filter', apply_hamming=True, hamming_method='msg', hamming_radius=2, hamming_ratio=15.0, hamming_ratio_metric='count', hamming_tie_break='lda', hamming_low_conf_perc=None, apply_confidence_filter=True, confidence_filter_method='percentile', confidence_value=90, confidence_filter_or_flag='flag', apply_cohesion=False, cohesion_min_cells=5, inplace=True, verbose=True)[source]

Complete debarcoding pipeline with transformation and postprocessing.

Runs the full debarcoding workflow: transformation -> intensity filtering -> debarcoding -> Hamming clustering -> confidence filtering -> (optional) cohesion.

Parameters:

  • adata (AnnData) – Annotated data object with mapped barcode channels.

  • transform_method ({'log', 'arcsinh'}, default 'log') – Transformation method to apply.

  • cofactor (float, default 10.0) – Cofactor for arcsinh transformation.

  • apply_intensity_filter (bool, default True) – Whether to apply intensity filtering.

  • intensity_method ({'rectangular', 'ellipsoidal'}, default 'rectangular') – Filtering method.

  • intensity_percentile (float, default 99.0) – For 'ellipsoidal': percentile threshold.

  • intensity_sum_low (float, optional, default 1.0) – For 'rectangular': lower percentile for channel sum.

  • intensity_sum_high (float, optional, default 99.0) – For 'rectangular': upper percentile for channel sum.

  • intensity_var_low (float, optional, default 1.0) – For 'rectangular': lower percentile for channel variance.

  • intensity_var_high (float, optional, default 99.0) – For 'rectangular': upper percentile for channel variance.

  • intensity_filter_or_flag ({'filter', 'flag'}, default 'filter') – 'filter' to remove cells or 'flag' to mark.

  • method ({'gmm', 'premessa', 'pc_gmm', 'manual'}, default 'pc_gmm') – Debarcoding method. PC-GMM recommended for most datasets (bimodal regime). PreMessa for unimodal regime. GMM for exploration only (assignments not constrained to valid patterns).

  • n_init (int, default 5) – GMM initializations. Used by: gmm, pc_gmm.

  • min_int (float, default 0.1) – Minimum intensity for GMM fitting. Used by: gmm, pc_gmm.

  • prob_thresh (float, default 0.5) – Probability threshold. Used by: gmm.

  • n_iter (int, default 2) – Normalization iterations. Used by: premessa.

  • thresholds (list of float, optional) – Manual thresholds. Used by: manual.

  • apply_hamming (bool, default True) – Whether to apply Hamming clustering.

  • hamming_method ({'msg', 'sphere'}, default 'msg') – Clustering method.

  • hamming_radius (int, default 2) – Max Hamming distance for neighbor search.

  • hamming_ratio (float, default 15.0) – Minimum ratio for remapping valid patterns.

  • hamming_ratio_metric ({'count', 'score'}, default 'count') – Ratio metric.

  • hamming_tie_break ({'no_remap', 'count', 'lda'}, default 'lda') – Tie-break method.

  • hamming_low_conf_perc (float, optional) – Only remap bottom N% confidence cells.

  • apply_cohesion (bool, default False) – Whether to compute cohesion scores after clustering.

  • cohesion_min_cells (int, default 5) – Minimum cells per cluster for cohesion computation.

  • apply_confidence_filter (bool, default True) – Whether to apply confidence-based filtering.

  • confidence_filter_method ({'threshold', 'percentile'}, default 'percentile') – Filtering method.

  • confidence_value (float, default 90) – Threshold or percentile value.

  • confidence_filter_or_flag ({'flag', 'filter'}, default 'flag') – 'flag' to mark or 'filter' to remove cells.

  • inplace (bool, default True) – Modify adata in place.

  • verbose (bool, default True) – Print progress messages.

Returns:

AnnData with debarcoding results.

Return type:

AnnData

Examples

>>> adata = debarcoding_pipeline(adata, method='pc_gmm')
>>> adata = debarcoding_pipeline(adata, method='pc_gmm', apply_hamming=False)
